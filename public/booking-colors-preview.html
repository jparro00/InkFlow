Theme tokens × themes

Every swatch resolves through CSS variables defined in the [data-theme=…] blocks above (mirror of src/index.css). Cell backgrounds use each theme's --color-bg; booking swatches use color-mix for the 7% alpha fill that DayView paints. Hex labels are read from getComputedStyle on the rendered swatch — they reflect what the browser actually paints, so a single edit to a theme variable updates every cell in the column.
